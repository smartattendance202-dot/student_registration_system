نظام تسجيل بيانات الطلاب

نظام ويب بسيط لتسجيل بيانات الطلاب مع إمكانية رفع الملفات

نظرة عامة

مرحباً بكم في نظام تسجيل بيانات الطلاب. هذا النظام مصمم لجمع البيانات الشخصية والأكاديمية للطلاب مع إمكانية رفع الوثائق المطلوبة مثل الصور الشخصية وكشف الدرجات.

مميزات النظام

  • تسجيل البيانات الشخصية والأكاديمية للطلاب
  • رفع الملفات المطلوبة (صورة شخصية وكشف درجات)
  • حفظ البيانات في قاعدة بيانات آمنة
  • واجهة مستخدم بسيطة وسهلة الاستخدام
  • نظام تسجيل دخول آمن

التقنيات المستخدمة

  • Python Flask (Backend)
  • HTML/CSS/JavaScript (Frontend)
  • SQLite/MySQL (Database)
  • Bootstrap 5 (لتنسيق الواجهة)
ملاحظة مهمة: هذا المشروع هو تطبيق ويب ديناميكي يتطلب خادم لتشغيله. الملفات الموجودة في هذا المستودع مخصصة للتوثيق والعرض التوضيحي. لتشغيل النظام فعلياً، يجب تنزيل الكود وإعداد بيئة الخادم.

كيفية التشغيل

  1. قم بتنزيل الكود من المستودع
  2. قم بإنشاء بيئة Python افتراضية
  3. ثبت المتطلبات باستخدام الأمر: pip install -r requirements.txt
  4. قم بتشغيل الخادم باستخدام الأمر: python run.py
  5. افتح المتصفح على العنوان: http://127.0.0.1:5000
